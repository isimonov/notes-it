# Base commands
| Command | Description |
|---------|-------------|
| `whoami` | current user |
| `pwd` | current working directory |
| `ls` | files list |
| `ll` | alias for `ls -l` |
# chown & chmod
| Command | Description |
|---------|-------------|
| `chown -R root:root *` | Changes the owner and group for all subfolders |
| `chmod -R 755 *` | Changes the access rights for all subfolders |
